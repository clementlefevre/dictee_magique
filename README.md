# How to update Questions
-  check the current levels of  questions *QUESTION_LEVEL_X* in `app/src/assets/text/text.content.json`.
- go the the `/input` folder.
- edit an existing file (e.g. 7.txt) or create a new one.
- type the list of words and save it using the naming rule : *level number*.txt ,for example `8.text`.
- save and commit the file.
- the current static webpage located [https://ladicteemagique.s3.amazonaws.com/index.html](https://ladicteemagique.s3.amazonaws.com/index.html) will be updated within 5 minutes.
